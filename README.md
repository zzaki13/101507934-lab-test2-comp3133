# Harry Potter Angular Application

## Student ID: 101507934

## App Description
An Angular application that fetches and displays Harry Potter character data using the HP API (https://hp-api.onrender.com/). Users can browse all characters, filter by Hogwarts house, and view detailed information about each character.

## Features Implemented
- Character List page displaying all Harry Potter characters with images and house names
- Filter by House feature with dropdown to filter characters by Gryffindor, Slytherin, Hufflepuff, or Ravenclaw
- Character Details page showing detailed information including name, house, actor, species, ancestry, wand details, and alive/deceased status
- Angular Material components for UI design
- Angular Signals for reactive state management
- Responsive grid layout with dark wizarding theme
- API Service using HttpClient to fetch data from HP API
- TypeScript interfaces for data structure
- Angular routing with route parameters
